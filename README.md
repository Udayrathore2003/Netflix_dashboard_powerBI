# Netflix_dashboard_powerBI
Developed an interactive Netflix Analytics Dashboard using Microsoft Power BI to analyze movies and TV shows by genre, country, release year, and ratings. Utilized Power Query, DAX, KPI cards, and interactive visualizations to transform raw data into actionable business insights.
